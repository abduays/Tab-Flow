# TabFlow Icon Requirements

For production, create PNG icons in the following sizes:
- icon16.png (16x16) - Toolbar icon
- icon32.png (32x32) - Extension manager
- icon48.png (48x48) - Chrome Web Store listing
- icon96.png (96x96) - High DPI displays  
- icon128.png (128x128) - Firefox Add-ons & Chrome Web Store

## Recommended Design:
- Wave/water flow motif (🌊) representing "TabFlow"
- Clean, modern gradient from blue (#58a6ff) to purple (#a371f7)
- Transparent background
- No text, just symbol

## Tools to Generate:
- Use Figma, Canva, or https://www.iconifier.net/
- Or run: `npx pwa-asset-generator ./src/icon.svg ./icons --background transparent`
